
## 16bit x 16bit Combinational signed multiplier
![multiplier](./docs/signed_multiplier.jpg) 

## Installation

Install iverilog and gtkwave
```bash
make install
```

## Run simulation
To compile design
```bash
make compile
```

To run the simulation
```bash
make simulate
```

To view simulation
```bash
make view_simulation
```

To delete files made by simulator
```bash
make clean
```
