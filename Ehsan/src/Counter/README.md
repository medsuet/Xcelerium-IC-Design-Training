# Installation
Install iverilog and gtkwave
```bash
make install
```

# Run Simulation
To compile design
```bash
make compile
```

To run the simulation
```bash
make simulate
```

To view simulation
```bash
make view_simulation
```

To delete files made by simulator
```bash
make clean
```
