## Installation

Install gcc compiler 
```bash
  sudo apt install gcc
```

Building Essential
```bash
  sudo apt install build-essential
```


## Run C files
To compile a C file, you can use the `gcc` compiler.

Making object file
```bash
  gcc -c (filename).o (filename).c
```
Making Executable
```bash
  gcc -o (filename) (filename).o
```
Running the Executable

```bash
  ./filename
```
